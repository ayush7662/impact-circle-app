

# 🏌️ Impact Circle App

A full-stack subscription-based web platform combining **score tracking, draw system, admin control, and charity-based engagement**.

## 🧩 Features

### 👤 User Features
- Signup / Login system
- Add golf scores (last 5 scores only)
- View dashboard
- Subscription-based access control
- Participate in monthly draw

### 💳 Subscription System
- Monthly & yearly plans
- Powered by Stripe integration
- Active subscription required for draw eligibility

### 🎲 Draw System
- Random monthly draw engine
- Matches user scores with generated numbers
- 3-match, 4-match, 5-match reward tiers
- Jackpot rollover logic

### 🛠️ Admin Panel
- View all users
- Run monthly draw manually
- Manage charities
- View winners list

---

## 🛠️ Tech Stack

- Frontend: Next.js
- Database: Supabase
- Payments: Stripe
- Deployment: Vercel
